# Process Pool Copy Program

## Instrucciones De Uso

### Compilación

Estando en la raíz del proyecto, ejecute el comando `sh compile.sh`.

### Ejecución

Llame al ejecutable `main.out` que se encuentra en la carpeta `bin`. Este ejecutable recibe 2 argumentos:
1. Directorio fuente.
2. Directorio destino. No es necesario que exista desde antes.

Tome en cuenta que las rutas se tomarán relativas desde donde se haga la llamada al ejecutable `main.out`.
